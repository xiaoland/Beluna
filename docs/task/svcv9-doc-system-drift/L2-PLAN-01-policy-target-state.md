# L2 PLAN 01 - Policy Target State

## Goal
Define the exact post-remediation policy state across `00/10/20/30/40` and `AGENTS.md`.

## Target State Contracts
1. Read strategy contract:
- "guided by context" language is primary.
- any order list is advisory, not mandatory ritual.

2. Canonical semantics contract:
- canonical product/domain semantics owned in PRD (`docs/10-prd`).
- `docs/00-meta/concepts.md` keeps cross-layer operational ontology and governance terms.

3. Unit TDD admission contract:
- explicit statement whether Beluna keeps all-unit Unit TDD intentionally, or narrows to hard-unit-only.
- rationale must be documented if intentionally diverging from V8/V9 admission philosophy.

4. AGENTS scope contract:
- AGENTS keep durable operating constraints.
- volatile capability snapshots are either slimmed or relocated to explicitly non-authoritative artifacts.
- risky/reference-sensitive changes require pre-execution restatement protocol.

5. Doc lifecycle contract:
- promotion rule remains.
- demotion/removal rule is explicit and operationalized.
