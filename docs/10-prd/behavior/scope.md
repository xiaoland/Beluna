# Scope

## In Scope

1. Product behavior for continuity, endpoint interaction, and operational trust claims.
2. User-observable workflow expectations for operators and endpoint integrators.
3. Product-level rules/invariants that constrain downstream technical realization.

## Out Of Scope

1. Internal module choreography and ownership contracts.
2. Wire-level transport internals beyond observable integration guarantees.
3. Runtime mechanism details that belong to TDD/deployment layers.
