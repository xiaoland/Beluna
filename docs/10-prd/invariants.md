# Product Invariants

1. Natural language is the protocol across cognition and body affordances.
2. Core runtime boundaries remain explicit: Cortex, Stem, Continuity, Ledger, Spine, AI Gateway, and Body endpoints.
3. Cortex cognition cycle is admitted by ticks; senses are buffered between admitted ticks.
4. Continuity persists cognition state and applies deterministic guardrails.
5. Act dispatch path is ordered and serial through `Continuity -> Spine`.
6. Descriptor identity uses fully qualified signal IDs (`endpoint_id/neural_signal_descriptor_id`).
7. Core is the single owner of runtime observability export.
8. Configuration contract is defined by typed config structures and boundary validation.
